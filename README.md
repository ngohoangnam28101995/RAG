0. **PREPARE:**
   - **Create virtual environment:**
     ```bash
     virtualenv --python=3.12 env-3.12
     ```
   - **Activate the environment:**
     ```bash - windows
     .\env-3.12\Scripts\activate
     ```
     ```bash - mac
     source env-3.12/bin/activate
     ```
   - **Install the requirements:**
     ```bash
     pip install -r requirements.txt
     ```
   - **Start application**
     ```bash
     python main.py
     ```
